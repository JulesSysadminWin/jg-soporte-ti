JG Soporte TI - Versión v11 Magistral

Qué trae esta versión:
- Diseño oscuro profesional, limpio y más elegante.
- Mejor experiencia móvil y escritorio.
- Animaciones suaves e interactividad sin depender de librerías externas.
- Menú responsive.
- Filtros de servicios.
- Tabs de precios.
- Canvas de partículas sutil.
- Páginas internas por servicio.
- Página de SEO técnico conservada, pero sin saturar la portada.
- Se retiró el bloque administrativo que no debía mostrarse en la web.
- Sin site.webmanifest, para evitar que aparezca "instalar aplicación".

Cómo subir:
1. Descomprime el ZIP.
2. Entra dentro de la carpeta descomprimida.
3. Copia todos los archivos y carpetas de adentro.
4. Pégalos en C:\Proyectos\JG-Soporte-TI\jg-soporte-ti.
5. Reemplaza archivos cuando Windows pregunte.
6. En VS Code Terminal:
   git status
   git add -A
   git commit -m "Actualizar web JG Soporte TI v11"
   git push origin main

Importante:
- No subir la carpeta completa; subir/copiar el contenido interno.
- El archivo CNAME debe quedar con: www.jgsoporteti.com
