Servicio formal

Se configura el sistema y las aplicaciones necesarias para dejar el equipo listo para trabajar, estudiar o usar en casa.

  • No se incluyen licencias piratas ni software de pago.
  • Se puede configurar Office si tienes cuenta/licencia válida.
  • Se instalan drivers principales.

Precio referencial

La instalación/configuración de Office inicia desde S/ 40. Windows con drivers inicia desde S/ 80 si requiere instalación limpia.

El precio depende del alcance y del estado del equipo.
